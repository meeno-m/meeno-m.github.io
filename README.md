## ☕ About Me

I'm a designer-developer with **10+ years** of work across **government, non-profits, and UN organizations** — UNICEF, UNCTAD, and UNDP. I take products from the first wireframe to the last line of code: front-end development, product design, and the data to prove it worked.

Computer Science degree in Kabul → master's in Information Systems in Japan → developer roles across Asia → MBA in Business Analytics and analyst work in the U.S. The through-line is **range**: fluent enough to code it, disciplined enough to design it, analytical enough to know if it worked.

This repo is my personal portfolio — a single self-contained `index.html`, built by hand, deployed on GitHub Pages.

## 🚀 Run it locally

```bash
# clone
git clone https://github.com/meeno-m/meeno-m.github.io.git
cd meeno-m.github.io

# open it — that's it, no build needed
open index.html          # macOS
# or just double-click index.html
```

**Deploy:** push to `main`, then in **Settings → Pages** choose *Deploy from branch → main → / (root)*. Live in a minute.
